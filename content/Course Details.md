- Structure changed over sems.
- Generative elements moved to separate course - GenAI.
- Focus on discriminative models.
- Why do things this way? Why not directly teach about LLMs?
	- LLMs are made up of foundational building blocks, without which they cannot exist.
	- This course teaches about these building blocks - autoencoders, transformers, etc.
	- Without an understanding of these building blocks, you may try to frame a problem which could have been solved using them as a generative problem, leading to a subpar solution.
- Exams: mix of objective, descriptive and solving questions.
- Assignments:
	- Will be a group assignment.
	- Technically two, but think of them as one contiguous one.
	- First one - reading and presenting paper.
	- Second one - implementing it and REPRODUCING results.
		- Evaluation for implementation - 15 min presentation and code check by TAs.
	- Difficult part about implementation - training.
		- Need to use intuition to decide hyperparameters.
	- Resource constraints are an issue. Need to work within them.
		- Options:
			- Train the model for just one epoch.
			- Try to figure things out and simplify them (DO THIS). 
- Books:
	- Second one is primary.
	- Third one is dense, but most up to date.
	- To brush up: Hands on Machine Learning
